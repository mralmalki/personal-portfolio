Image Editor

Technologies I Used:

  • TwelveMonkeys ImageIO plugin for PPM file support
  • Maven build automation and dependency manager
  • A little bit of MVC

Project Description

Image Editor was a project that I completed on my own during fall 2022. It is a GUI-based application for image editing in Java, using the Swing framework. The project for Assignment 8 in the C212 FA22 course is to develop a GUI-based image editor similar to a project produced in the previous assignment but with improved user experience through the use of the Swing framework in Java. Students are expected to adapt existing solutions into a broader project, use tests to demonstrate the quality and behavior of their work, and become proficient in developing sophisticated programs with GUI elements in Java. The project involves building the front-end GUI for the application using a starter code and connecting the functionality of its interface elements to the behind-the-scenes logic. The technologies used in the project include the TwelveMonkeys ImageIO plugin for PPM file support, Maven build automation and dependency manager, and a little bit of Model-View-Controller.

What I learned

From this project, I learned to develop a sophisticated program with GUI elements in Java, using the Swing framework. I learned to adapt existing solutions into a broader project, and use tests to demonstrate the quality and behavior of my work. I gained knowledge about technologies such as the TwelveMonkeys ImageIO plugin for PPM file support, Maven build automation and dependency manager, and the Model-View-Controller (MVC) design pattern. I also learned to work with starter code, understand how it works, and build off of it. Additionally, I was able to build the front-end GUI for an image editor application, which allowed the user to choose an image, apply a transformation, save that image, and repeat the process in an easy, visualizable way. Finally, I learned to follow Java conventions and write clean and well-documented code.
